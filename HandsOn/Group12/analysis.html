Encuestas de calidad de vida (encuestas2014.csv)

Resultados de las 2500 encuestas de calidad de vida realizadas por el Ayuntamiento de Madrid. El Ayuntamiento no ofrece la explicación de cada columna pero sí el cuestionario realizado tal cual. A partir de dicho cuestionario, se ha deducido qué podría ser cada campo.

Campos que no son preguntas

Column Type Comments Problems
NUNCUESTI Integer Unique
SECCION Integer Not unique. Code of "sección censal"
BARRIO String Not unique Encoding problems with some characters
CODBARRIO Integer Not unique. Constrain: fields with the same CODBARRIO value must have the same BARRIO value Constrain not checked
DISTRITO Integer Not unique
SER Integer Not unique Not sure what this mean. Probably related to "Servicio de Estacionamiento Regulado" but not sure
ENCUESTADOR Integer Not unique Not sure what this field means. Probably is the ID of the person who do the questionnaire
RUTA Integer Not unique Not sure what this field means.
PESO Decimal Not unique. Decimal point is a comma, not a point Not sure what this field means.

Campos que son preguntas

Column Type Comments Problems
P1X ~ P5X (where X is a letter) Number Not unique. Values 1, 2, 3, 4, 5, 8 or 9. Each column matches with each sub-question of the questionnaire
P6XLIT ~ P7XLIT (where X is a letter) String Not unique Can have spelling errors
P6X ~ P7X (where X is a letter) Number. Can be NULL. Not unique. It is like a "serialized" version of P6XLIT and P7XLIT
P8X ~ P8X (where X is a number from 1 to 20) Number. Not unique. Answers to each sub-question of question 8
P9X1 ~ P9X1 (where X is a number from 1 to 25) Number. Values from 0 to 12. Not unique. Answers to each "Satisfacción" sub-question of question 9
P9X2 ~ P9X2 (where X is a number from 1 to 25) Number. Values 1, 2, 8 or 9. Not unique. Answers to each "¿Lo ha utilizado...?" sub-question of question 9
P9X3 ~ P9X3 (where X is a number from 1 to 25) Number. Values 1, 2, 3, 8 or 9. Not unique. Answers to each "¿Su número le parece excesivo...?" sub-question of question 9
P10X ~ P10X (where X is a number from 1 to 13) Number. Can be NULL. Not unique. Relevant value is boolean-like: NULL (false) or NOT NULL (true)
P11X ~ P18X (where X is other number) Integer. Not unique. Answers to sub-question X of the question. Values specified in the questionnaire.
P19X ~ P19X (where X is a number from 1 to 12) Number. Can be NULL. Not unique. Relevant value is boolean-like: NULL (false) or NOT NULL (true)
P20X ~ P20X (where X is other number) Integer. Values can be 1, 2 or 9.
P21X ~ P29X (where X is other number) Integer. Not unique. Answers to sub-question X of the question. Values specified in the questionnaire.
